Важные горячие клавиши

Три важные комбинации клавиш, которые должен знать каждый.

Горячие клавиши — сочетание определенных клавиш на клавиатуре, нажатие которых дает команду программе. То есть зажимая эти клавиши, можно совершать действия, не используя мышку

Горячие клавиши для смены раскладки клавиатуры

Для смены языка используется одна из следующих сочетаний клавиш (это зависит от настроек компьютер)

Alt + Shift

Или:

Ctrl + Shift

Копирование и вставка текста

Для копирования используйте комбинацию:

Ctrl + C

Для вставки используйте комбинацию:

Ctrl + V
